# Evilz blog

Migration du blog vers **Astro** + **Tailwind CSS** en s'appuyant sur **AstroWind**.

## Développement

```bash
npm install
npm run dev
```

## Vérification

```bash
npm run check
npm run build
```

## Déploiement

Le dépôt contient un workflow GitHub Actions qui construit le site et le publie sur GitHub Pages.
