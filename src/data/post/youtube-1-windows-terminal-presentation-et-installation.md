Dans cette vidéo, Bumble dev se concentre sur la prise en main de Windows Terminal, son installation et les premiers réglages utiles. Le sujet est abordé avec un angle très pratique, orienté installation, configuration et retour d'expérience sur un poste de dev moderne sous Windows.

On y retrouve l'idée centrale de la chaîne : rendre l'environnement de travail plus agréable et plus efficace au quotidien. Que vous utilisiez WSL, Windows Terminal ou Windows 11, la vidéo aide à comprendre l'intérêt de ces outils et les points d'attention avant de se lancer.
