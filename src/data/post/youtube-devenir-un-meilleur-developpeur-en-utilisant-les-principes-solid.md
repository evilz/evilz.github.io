Cette vidéo propose une introduction pragmatique aux principes SOLID et à leur impact sur la qualité d'un code maintenable. Bumble dev explique pourquoi ces principes restent utiles bien au-delà de la théorie, notamment pour garder une base de code saine quand un projet grandit.

Le propos s'adresse autant aux développeurs débutants qu'aux profils plus expérimentés qui veulent remettre un peu d'ordre dans leur façon de concevoir des classes, des interfaces ou des modules. On ressort avec une grille de lecture simple pour repérer les points de friction dans un code existant.
