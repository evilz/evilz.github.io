Cette vidéo est centrée sur la création d'une API REST avec F# et une approche fonctionnelle. Bumble dev y montre comment F# peut offrir une approche plus concise et plus expressive, tout en restant très pratique pour construire des applications ou explorer une architecture.

La vidéo s'adresse à la fois aux curieux du paradigme fonctionnel et aux développeurs .NET qui veulent élargir leur boîte à outils. Elle met en avant des idées réutilisables, même si l'on ne code pas tous les jours en F#.
