Bumble dev présente ici une sélection d'extensions VS Code utiles pour gagner du temps en développement. La vidéo met l'accent sur les gains de productivité que l'on peut obtenir avec un bon outillage, sans tomber dans une configuration inutilement compliquée.

Au-delà de la démonstration, la vidéo donne aussi des idées concrètes pour adapter son environnement à sa manière de travailler. C'est particulièrement utile si vous aimez optimiser votre poste de développement ou découvrir de nouveaux outils autour de VS Code, du terminal ou de l'écosystème .NET.
