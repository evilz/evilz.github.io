Cette vidéo présente une chaîne d'intégration continue simple autour de GitHub et Docker. Bumble dev relie les outils entre eux pour montrer comment automatiser un flux de travail sans complexifier inutilement le projet.

Le sujet est particulièrement utile pour les développeurs qui veulent passer d'un projet local à une approche un peu plus industrialisée. On y voit l'intérêt d'un pipeline simple, reproductible et facile à faire évoluer ensuite.
